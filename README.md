## 📌 Project Overview

This project is a **Natural Language Processing (NLP) based AI Chatbot** built using Python.  
It uses **TF-IDF Vectorization** and **Cosine Similarity** to understand user queries and generate the most relevant response from a predefined dataset.

This project was developed as part of the **CodeAlpha Artificial Intelligence Internship**.

---

## 🎯 Objective

- Build a simple AI chatbot using NLP techniques  
- Understand text preprocessing and similarity matching  
- Implement TF-IDF and Cosine Similarity  
- Create a working conversational system  

---

## 🚀 Features

✔ NLP-based chatbot  
✔ TF-IDF text vectorization  
✔ Cosine similarity matching  
✔ Handles greetings and AI-related queries  
✔ Lightweight and runs in Google Colab  
✔ Beginner-friendly AI project  

---

## 🧠 How It Works

1. User enters input  
2. Text is preprocessed (lowercase, tokenization, lemmatization)  
3. TF-IDF converts text into numerical vectors  
4. Cosine similarity finds best matching sentence  
5. Bot returns the most relevant response
