## 📌 Project Overview

This project is a simple **AI-powered chatbot** built using Python and NLP techniques.  
It uses **TF-IDF Vectorization** and **Cosine Similarity** to match user input with the most relevant response from a predefined dataset.

This project was completed as part of the **CodeAlpha AI Internship**.

---

## 🎯 Objective

- Build a basic conversational AI system  
- Learn NLP preprocessing techniques  
- Understand similarity-based response matching  
- Implement TF-IDF and cosine similarity  

---

## 🛠 Technologies Used

- Python 🐍  
- NLTK  
- Scikit-learn  
- NumPy  
- Google Colab  

---

## 🧠 How It Works

1. User enters a message  
2. Text is cleaned (lowercase, tokenization, lemmatization)  
3. TF-IDF converts text into vectors  
4. Cosine similarity finds closest match  
5. Bot returns best response
